CJ's Project 1

this list will go through each requirement and how I have met it.

  1. My site serves others, as it educates them on how to purchase and maintain a car.
  2. My site is original. all text was typed by me over multiple days. All images used are under a free for commercial use license or are photos I took myself.
  3. My site has 5 pages as is apparent on the nav bar. the home page is named index.html.
  4. My site includes a nav header for easy navigation.
  5. My site includes navigation links in the header and some links scroll the page to different sections, such as in the inspection page.
  6. My site uses media queries to determine the margin on the sides of pages that have more whitespace than others, such as the pricing page.
  7. My site uses a blankent style.css that handles all style that is shared between pages. My site then uses individual stylesheets for each page of the site.
  8. My site includes 5 images. All images on the site have an alt that accurately describes the content of the image.
  9. My site declares a favicon on every page.
  10. My site places all non html sources into fodlers. those folder being named js, css, and images.
  11. My site uses no frameworks. It uses html, css, and basic js to do a simple mathematical calulation.
  12. My site fully passes Nu Validator for all stylesheets and html files.
  13. My site contains strong axes of alignment. All non header/hook elements are left aligned with a margin from the edge of the page.
  14. My site has content with adequete space and no text bleeds against its bounding box.
  15. My site uses two different font faces. One is for the headers, and the other is for the text content.
  16. My site includes two froms. One which does nothing, and another that performs a mathematical calculation.
  17. My site includes multiple links to external sites in the modifications page.
  18. My site contains redable source code, consistent formatting, and meaningful naming conventions utilizing kebab-case identifiers.
  19. My site contains an animation of the logo on the home page.
  20. My site has been shared with twodee on github.
  21. My site contains no files with non alphanumeric or uppercase characters.
  22. My site is available via hhtp and https.
  23. My site is served over port 443.
  24. My site redirects http requests to https requests.
  25. My site has a git repository that shows a steady flow of commits with an increasing frequency as the deadline was approached.
  26. My site contains a page called report.html that contains an itemized breakdown of how I believe I have met all of the expectations above.
